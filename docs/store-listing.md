## 概要 / 説明文（本文）

**タスクを「管理」するのではなく、今日「決着」させるためのアプリです。**

やることリストは、放っておくと際限なく増えて、どれも進まなくなります。やっとこ は逆の発想。**入れられる数に上限**を設け、今日やることだけに絞り込みます。

**使い方はシンプル。**
1. ＋ でやることを **BOX** に入れる（音声入力にも対応）。
2. BOX のカードを **右スワイプで TODAY（今日やる）／左スワイプで LATER（あとで）** に仕分け。
3. TODAY は今日中に決着。残り時間が夜に近づくほど赤くなり、背中を押します。
4. 夜になったら **今日の精算** —「明日もTODAY / LATERへ / 完了 / 削除」で、未完了を溜めずに片づけます。

**溜めさせないための仕組み。**
- BOX / TODAY / LATER にはそれぞれ**上限**があり、満杯になったら整理を促します。
- LATER は「墓場」ではなく**未来の TODAY**。開始日が来たら自動で TODAY に戻ります。
- 3 日連続で手つかずの TODAY は、自動で LATER に戻して頭の中を軽くします。
- アプリアイコンの**バッジ**が、TODAY の未完了数を静かに教えてくれます。

**あなたのデータは、あなたの端末の中だけに。**
- クラウド同期・ログイン・アカウント登録はありません。
- 入力したタスクは端末内にのみ保存され、開発者に送信されることはありません。

**App 内課金（広告なし）**
- **Pro**: BOX / TODAY / LATER / ジャンルの上限を拡張。
- **ブースト（¥100 の買い切り）**: BOX+5 / TODAY+2 / LATER+5 を恒久的に追加。
- 無料のままでも中心機能はすべて使えます。広告は表示しません。

先延ばしをやめたいのに、ToDo アプリがまた別の「積みタスク」になっていませんか。やっとこ で、今日に決着を。
